# Koala Hacks UI Library Changelog (v1.0.x -> v2.0.x)
This changelog includes the changelog for the builder of the library.

- ### **\[=\] Completely changed how the library works**
  - \[+\] Added a slider action
  - \[-\] Themes was completely removed, planned to be added in the future or for a major update
  - \[-\] GetSupportedVersions() was completely removed for the builder on v2.0.x, planned to use seperate builders for each version, or use a builder from another version instead of using 1 builder file
  - \[=\] Every function was renamed, changed how it was coded or completely changed
  
- ### **\[=\] Overhauled the UI of the library**
  - \[+\] Made the UI draggable
  - \[+\] Added a subtitle in the UI for tabs
  - \[~\] Changed the toggle action to a switch like design instead of tweening from the color red (or $ToggleColorOff theme) to color green (or $ToggleColorOn theme)
  - \[~\] Designed the UI to have a more smoother edges and have a "modern look"

## Symbols and meanings
- \[+\] <- Added a new feature
- \[~\] <- Changed an existing feature
- \[-\] <- Removed a previously existing feature
- \[=\] <- Mixed or broad
